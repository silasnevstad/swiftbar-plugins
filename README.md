# SwiftBar Plugins

Small, local-only SwiftBar plugins.

## Install
1. SwiftBar → Preferences → Open Plugin Folder
2. Copy a plugin script from `plugins/<name>/` into that folder (or symlink it)
3. Ensure it is executable: `chmod +x <file>`

## Plugins
- [mail-unread](plugins/mail-unread/README.md) — Unread count from Apple Mail Inbox (all accounts, local-only, lightweight)
